# Design System

Baseado no design theme do Stitch `Adicionar Despesa`.

## Direção visual

- Norte criativo: `The Sovereign Sanctuary`
- Atmosfera: premium financial lounge, superfícies sobrepostas e sem divisórias duras
- Regra principal: evitar linhas de 1px; preferir mudança tonal e camadas

## Tipografia

- Display e headlines: Manrope
- Texto corrido: Inter

## Tokens principais

- `primary`: `#001949`
- `primary-bright`: `#0040A1`
- `primary-soft`: `#DAE2FF`
- `secondary`: `#1B6D24`
- `secondary-soft`: `#A0F399`
- `danger-soft`: `#FFDBD1`
- `background`: `#F8F9FA`
- `surface-low`: `#F3F4F5`
- `surface`: `#FFFFFF`

## Regras de implementação

- Use gradientes suaves para CTAs principais
- Trate cards como superfícies elevadas por cor e blur, não por sombra pesada
- Preserve cantos arredondados e espaçamento amplo entre blocos
